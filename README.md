# ICONIC GT — Site (v3) — GitHub Pages

Ce dossier contient un site statique **bilingue FR/EN** pour ICONIC GT (atelier privé).

## Déploiement sur GitHub Pages (repo: `iconic-gt`)

1. Sur GitHub, créez un dépôt **public** nommé `iconic-gt`.
2. Uploadez **tout le contenu** de ce dossier à la racine du dépôt (vous devez avoir `index.html` à la racine).
3. Allez dans **Settings → Pages** :
   - Source: **Deploy from a branch**
   - Branch: `main` (ou `master`)
   - Folder: `/ (root)`
4. GitHub affichera l'URL du site. Elle sera en général :
   `https://<votre-user>.github.io/iconic-gt/`

## Modifier la langue
- Le bouton en haut à droite bascule FR/EN.
- La langue choisie est mémorisée dans le navigateur (localStorage).

## Remplacer des images
Les images sont dans `assets/images/`.
Vous pouvez remplacer un fichier en gardant **le même nom** (ex: `r129_cover.jpeg`) pour qu'il soit automatiquement remplacé sur le site.

## Confidentialité / Atelier privé
Le site indique une zone (Saint‑Raphaël / Cannes) sans publier d’adresse précise.
